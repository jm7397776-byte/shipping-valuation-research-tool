사용 방법
  1. 기본 표본은 자동으로 열립니다.
  2. 선종 분류 또는 주력 분류를 누르면 회사 목록이 바뀝니다.
  3. 회사 이름을 누르면 오른쪽 대시보드가 해당 회사 중심으로 바뀝니다.
  4. 주력 분류는 가스·Dry bulk·Container·Tanker를 따로 봅니다.
  5. 선종 분류는 가스·벌크·컨테이너·탱커·일반화물·오프쇼어·여객·기타까지 함께 봅니다.
  6. 슬라이더로 주력 선종 기준을 조절합니다.
  7. IMO 단위 선대 원장은 선대 템플릿 형식으로 불러옵니다.
  8. 입력 템플릿에 재무값을 채운 뒤 CSV로 불러옵니다.
  9. 상단 버튼은 먼저 미리보기를 열고, 필요할 때만 내려받습니다.
  10. 논문 작성 도우미에서 가설, 변수, 방법론, 문헌검색 링크를 확인합니다.
